initial_conditions.create_voronoi module

class CreateVoronoi(context: Context)[source]

Bases: object

Task to create Voronoi tessellation from given starting image.

Working location structure for a given context:

(name)
└── images
    ├── (name)_(image key 1).ome.tiff
    ├── (name)_(image key 2).ome.tiff
    ├── ...
    ├── (name)_(image key n).ome.tiff
    ├── ...
    ├── (name)_(image key 1)_(channel)_voronoi.ome.tiff
    ├── (name)_(image key 2)_(channel)_voronoi.ome.tiff
    ├── ...
    └── (name)_(image key n)_(channel)_voronoi.ome.tiff

Voronoi images are saved to the same directory as the input images with appended channel value and a “voronoi” label.

Create Voronoi task.

Loads image from working location. Creates boundary for Voronoi using binary dilation, then performs the Voronoi tessellation using the selected channel of the image.

  • iterations – Number of boundary estimation steps.
